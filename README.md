# HACKTHON

Jogo de perguntas e respostas com relação a educação financeira inspirada no duolingo.

## OBJETIVO

O objetivo do nosso projeto

## SOBRE O PROJETO

### Plano Atual

Jogo simples de perguntas e respostas sobre educação financeira

#### Escopo Inicial:
- 4 niveis de dificuldade;
- Sistema de XP e classificação (experiencia);
- Mapa inicial;
- Atividades futuras;
- Missões (Acrescimo de xp);
- Planos futuros para o usuario;

### Plano Futuro

#### Escopo:
- Aprendizado de Investimentos com analise de graficos;
- Aplicação de agente de AI para orientação;
- Deep Learning de serie temporal para previsão de investimentos;
- Deep Learning de classificação de nivel de entendimento economico do usuario;
